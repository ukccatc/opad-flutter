# 🔥 Создание нового Firebase проекта для OPAD

## Шаг 1: Создание проекта в Firebase Console

1. **Откройте Firebase Console:**
   https://console.firebase.google.com/

2. **Нажмите "Add project" (или "Создать проект")**

3. **Введите название проекта:**
   - Рекомендуемое название: `opad-web` или `opad-flutter`
   - Или любое другое название на ваш выбор
   - **Важно:** Название проекта будет использоваться для генерации URL

4. **Настройте Google Analytics (опционально):**
   - Можно включить или пропустить
   - Для простого хостинга не обязательно

5. **Дождитесь создания проекта** (обычно несколько секунд)

## Шаг 2: Включите Firebase Hosting

После создания проекта:

1. В боковом меню выберите **"Hosting"**
2. Нажмите **"Get started"** (или "Начать")
3. Следуйте инструкциям (можно пропустить начальную настройку)

## Шаг 3: Свяжите проект с локальной директорией

После создания проекта выполните в терминале:

```bash
cd "/Users/macbookpro/Git Actions/flutter-opad"
firebase use --add
```

**Что делать:**
1. Выберите созданный проект из списка (используйте стрелки и Enter)
2. Введите алиас: `default` (или просто нажмите Enter)
3. Файл `.firebaserc` обновится автоматически

## Шаг 4: Проверка

Проверьте, что проект выбран:

```bash
firebase use
```

Должен показать ваш проект.

## Шаг 5: Деплой

После настройки проекта:

```bash
flutter build web --release
firebase deploy --only hosting
```

## Важные моменты

- **Project ID** будет автоматически сгенерирован Firebase (например: `opad-web-12345`)
- **URL приложения** будет: `https://YOUR_PROJECT_ID.web.app`
- Можно настроить кастомный домен позже в Firebase Console → Hosting

## Если возникли проблемы

- Убедитесь, что вы авторизованы: `firebase login`
- Проверьте список проектов: `firebase projects:list`
- Если проект не виден, обновите страницу в Firebase Console
